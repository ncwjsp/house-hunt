## Installation

_To use this project locally_

1. Clone the repo
   ```sh
   git clone https://github.com/ncwjsp/house-hunt.git
   ```
2. CD into the directory
   ```sh
   cd house-hunt
   ```
3. Install packages
   ```sh
   npm install
   ```
   or with Bun
   ```sh
   bun install
   ```
4. Run the project
   ```sh
   npm run dev
   ```
   or with Bun
   ```sh
   bun dev
   ```
